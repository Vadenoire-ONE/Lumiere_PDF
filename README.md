# Lumiere PDF

Простая и лёгкая программа для работы с PDF на Python.

## Возможности
- Просмотр PDF с панелью миниатюр (клик по миниатюре — переход к странице)
- Масштаб, навигация по страницам, прокрутка колесом мыши, Ctrl+колесо — зум
- Объединение нескольких PDF в один
- Разделение PDF по диапазонам страниц (`1-3,5,7-9`) или постранично
- Определение формата листа (A4, A3, Letter, ...) для каждой страницы
- Извлечение текста (всего документа или текущей страницы) в `.txt`
- Извлечение всех изображений из PDF в выбранную папку

## Установка
```powershell
pip install -r requirements.txt
```

## Запуск
```powershell
python lumiere_pdf.py
```

### Быстрый запуск в Windows
Дважды кликните по `Запустить Lumiere PDF.bat` — он автоматически:
1. создаст виртуальное окружение `.venv` (только при первом запуске),
2. установит зависимости из `requirements.txt`,
3. запустит программу без консольного окна.

### Полностью портативная сборка (один `.exe`, ничего ставить не нужно)
Дважды кликните по `Собрать exe.bat`. Скрипт:
1. создаст `.venv` и поставит `PyInstaller`,
2. соберёт проект по конфигу `lumiere_pdf.spec`,
3. положит результат в `dist\LumierePDF.exe`.

Получившийся `LumierePDF.exe` — **один файл** (~50 МБ),
содержит Python + PyMuPDF + Pillow + Tkinter.
Его можно скопировать на любую Windows 10/11 и запускать без установки.

## Горячие клавиши
| Клавиша | Действие |
| --- | --- |
| `Ctrl+O` | Открыть PDF |
| `←` / `→`, `PageUp` / `PageDown` | Предыдущая / следующая страница |
| `Ctrl++` / `Ctrl+-` | Увеличить / уменьшить масштаб |
| `Ctrl+колесо` | Масштаб |
| `Колесо` | Прокрутка |

## Зависимости
- [PyMuPDF](https://pymupdf.readthedocs.io/) — работа с PDF
- [Pillow](https://pillow.readthedocs.io/) — отображение страниц
- Tkinter — входит в стандартную поставку Python
